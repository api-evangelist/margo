# Margo (margo)
Margo is a Linux Foundation open standard initiative for interoperability at the edge of industrial automation ecosystems. Founded by ABB, Capgemini, Microsoft, Rockwell Automation, Schneider Electric, and Siemens, it defines mechanisms for interoperability between edge applications, devices, and orchestration software.
